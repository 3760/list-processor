# 打包验证指南

## 概述

本指南说明如何使用 PyInstaller 对工具进行打包，并在不同平台上进行验证。

## 打包命令

### macOS

```bash
cd 代码目录
pyinstaller main.spec --clean
```

打包完成后，输出文件位于：
```
dist/客户名单数据预处理工具/
├── 客户名单数据预处理工具.app  # macOS 应用
└── ...
```

### Windows

```bash
cd 代码目录
pyinstaller main.spec --clean
```

打包完成后，输出文件位于：
```
dist/客户名单数据预处理工具/
├── 客户名单数据预处理工具.exe  # Windows 可执行文件
└── ...
```

## 常见问题

### Q1: 打包后缺少 xxx 模块

**原因**：模块未被 PyInstaller 自动检测到

**解决方案**：在 `main.spec` 的 `hiddenimports` 中添加该模块：

```python
hiddenimports = [
    ...
    'missing_module',  # 添加缺失的模块
]
```

### Q2: 打包后找不到配置文件

**原因**：`datas` 配置不正确

**解决方案**：检查 `main.spec` 中的 `datas` 配置：

```python
datas = [
    ('config/app_config.yaml', 'config'),
    ('config/field_spec.yaml', 'config'),
    ('dict/data_dict.xlsx', 'dict'),
]
```

### Q3: macOS 打包后无法打开（已损坏）

**原因**：macOS 安全策略阻止未签名应用

**解决方案**：
```bash
# 临时允许运行（仅开发测试）
xattr -r -d com.apple.quarantine dist/客户名单数据预处理工具.app
```
