1. 工具简介

本工具用于在 CEM 系统投放前，对客户名单进行自动化预处理，包括：

  • 字段合规性检查：自动检查一线名单字段是否符合 CEM 规范

  • 数据字典上码：根据统一码表为字典类型字段自动添加 Code 列

  • 字典值合规校验：检查字典字段值是否在码表范围内

  • 名单内部重复检查：检查一线名单内部是否存在重复记录

  • 跨名单去重标注：在三方/HW 名单中标注是否已存在于一线/三方名单中

处理结果输出为 3 个独立 Excel 文件（一线/三方/HW 各一个），存放于自动创建的批次子文件夹中。

2. 环境要求

注意：本工具为单机可执行文件（.exe / .app），无需安装 Python 环境。

3. 启动方式

Windows

双击 名单处理工具.exe 即可启动。

macOS

双击 名单处理工具.app 即可启动。

首次启动如遇"无法打开"提示，请前往 系统设置 > 隐私与安全性 中点击"仍要打开"。

4. 操作步骤

步骤 1：准备输入文件

在使用工具前，请确认以下文件已准备就绪：

名单优先级：一线 > 三方 > HW。去重标注时，HW 名单会同时标注"是否已在一线名单"和"是否已在三方名单"。

步骤 2：加载文件

  1. 一线人员名单：点击「浏览」选择一线名单文件。若文件含多个 Sheet，会弹出 Sheet 选择对话框，默认已预选第一个 Sheet，直接点击确认即可。

  2. 三方/HW 名单（如有）：同上操作。

  3. 数据字典：点击「浏览」选择数据字典 xlsx 文件。工具会自动计算 MD5 并显示版本号。

  4. 字段规范：点击「导入」选择 CEM「属性导入模版」Excel 文件。工具会自动解析并更新 field_spec.yaml。

Sheet 选择技巧：多 Sheet 文件弹窗后，5 秒内无操作会自动使用预选的 Sheet。如需选择其他 Sheet，请在 5 秒内手动切换。

步骤 3：确认去重字段

工具会自动识别去重字段（默认按邮箱），识别结果显示在下拉框中。
若识别不正确，点击下拉框手动选择其他字段（如手机号）。

去重字段识别规则：

  1. 优先读取配置文件中预设的去重字段

  2. 自动匹配列名含 "email"/"邮箱"/"mail"/"e-mail" 的列

  3. 以上均失败时，需手动指定

步骤 4：选择处理模块

勾选需要执行的模块：

模块执行顺序：字段合规检查 → 名单内部重复检查 → 数据字典上码 → 字典值合规校验 → 跨名单去重标注。此顺序固定，不可调整。

步骤 5：设置输出目录

输出目录默认为一线名单所在文件夹。点击「浏览」可修改为其他位置。

步骤 6：开始处理

点击 「开始处理」 按钮。处理过程中：

  • 进度条实时显示当前进度

  • 日志区域显示各模块执行状态

  • 可随时点击「取消」中止处理（已生成的输出文件会被保留）

步骤 7：查看结果

处理完成后，界面顶部显示结果横幅：

  • 绿色：处理完成，无问题，可直接上传 CEM

  • 黄色：处理完成，但发现 X 条问题，建议修复后再上传

同时自动弹出结果查看器，可预览各 Sheet 内容。点击「打开输出目录」可直接定位到结果文件。

6. 常见问题（FAQ）

Q1：为什么原始数据 Sheet 里还有问题行？

A：工具将问题数据同时隔离输出到独立的「合规性检查结果」Sheet，但原始数据 Sheet 中保留全量数据，方便您对照原始值修改。问题行不参与后续模块处理（如上码、去重）。

Q2：重复名单结果在哪里查看？

A：v1.1.0 版本将重复信息合并到了「原始数据」Sheet 中，请查看以下三列：

  • _重复键值：去重字段值

  • _出现次数：该值出现次数

  • _内部去重结果："原始"（首次出现）/ "重复"（后续出现）/ "未知"（去重字段为空）

Q3：处理 100 万行数据需要多久？

A：标准配置电脑（8核/16G）约 2~5 分钟，具体取决于勾选的模块数量。Excel 写入是主要耗时环节。

Q4：可以只处理一线名单，不处理三方/HW 吗？

A：可以。三方和 HW 名单为选填项。仅加载一线名单时，跨名单去重标注模块会自动跳过。

Q5：字段规范导入失败怎么办？

A：常见原因：

  1. attr_code 重复：Excel 中属性 code 列有重复值，需修正后重新导入

  2. 必填字段缺失：有字段标记为必填但 attr_code 为空，需补全

  3. 格式不符：请确认使用的是 CEM 系统「属性导入模版」原始格式

Q6：字典文件更新后工具如何识别？

A：工具在本次会话内检测到您选择了与之前 MD5 不同的字典文件时，会弹窗提示"检测到字典文件已变更"。点击"是"即可使用新版本。

Q7：macOS 上按钮样式显示异常？

A：macOS 上 Qt 按钮的圆角样式可能不生效，但不影响功能。此为 macOS Qt 已知限制。

Q8：处理过程中可以取消吗？

A：可以。点击「取消」按钮，确认后处理线程会被中断。已生成的输出文件会被保留，但可能不完整。
